Este  mini-projetotem  como  objetivo  mostrar,  de  forma  prática  e  aplicada,  como  a Linguagem Python e a biblioteca NumPy podem ser usadas para extrair valor de dados brutos em um contexto de negócio real, da área de Marketing.A  situação  simulada  envolve  uma  plataforma  de  e-commerce  que  coleta  diversas informações   sobre   o   comportamento   dos   usuários,   como   número   de   visitas,   tempo   de navegação,  itens  adicionados  ao  carrinho  e  valores  de  compra,  mas  ainda  toma  decisões  de marketinge produto com base em intuição e métricas superficiais.Ao   longo   do   projeto,   o   você   aprenderá   a   gerar   um   conjunto   de   dados   fictícios representando centenas de usuários, cada um com métricas relevantes para análise. Em seguida, aplicará estatísticas descritivas como média, mediana, desvio padrão e identificação de valores extremos para entender o comportamento geral da base de clientes. Também  são  feitas  segmentações  com  indexação  booleana  para  identificar  perfis distintos,  como  clientes  de  alto  valor  e  visitantes  que  não  compram,  permitindo  descobrir oportunidades de conversão e personalização de campanhas.Outro ponto fundamental é o estudo de correlação entre variáveis, verificando relações como o impacto do tempo gasto no site e da quantidade de itens no carrinho sobre o valor final da   compra.   Essa   análise   permite   enxergar   padrões   de   comportamento   que   influenciam diretamente a receita e fornecem dados concretos para decisões estratégicas.
O projeto demonstra, de forma acessível e prática, como transformar dados em insights acionáveis para marketing, vendas e experiência do usuário (UX). Ao final, você compreenderá como usar o NumPy para realizar cálculos estatísticos, manipular matrizes, explorar dados e gerar conclusões  que  impactam  o  negócio,  desenvolvendo  habilidades  essenciais  para  quem  quer atuar em Data Science ou na área de dados de um modo geral.
